## 🚀 Features

- [x] 🚫 **Ad-Free** — zero interruptions, just music.
- [x] 🦀 **Plugin System** — secure, auto-updating `.bex` plugins for endless sources.
- [x] 🎤 **Karaoke Lyrics** — time-synced with manual offset adjustment.
- [x] 🎛️ **Equalizer** — built-in EQ and customizable crossfade.
- [x] 🔄 **Smart Replace** — auto-recovery if a playlist track goes dead.
- [x] 📊 **Last.fm Scrobbling** — logs listening history with offline cache.
- [x] 🎮 **Discord Rich Presence** — show current track on Discord.
- [x] 🌍 **Global Charts** — daily updated charts from installed plugins.
- [x] 💾 **Backup & Restore** — export/import library and settings via JSON or M3U.
- [x] 🤖 **AI Recommendations** — smarter suggestions via Last.fm/plugin data.
- [x] 👥 **Listen Together** — real-time sync across devices. Create or join a room with a 6-character code; host controls playback for all listeners.

---

## 👥 Listen Together

Sync playback with friends in real time — no account needed.

1. Open the sync sheet from the player screen.
2. **Host:** tap *Create a Room* → share the 6-character code.
3. **Guest:** enter the code → tap join.

Once connected, the guest's track, position, and play/pause state stay locked to the host. Works across different queues — the guest loads the host's track automatically even if it's not in their library.
